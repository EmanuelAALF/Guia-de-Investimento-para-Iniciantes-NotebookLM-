# Guia-de-Investimento-para-Iniciantes-NotebookLM-
O "Guia de Investimentos para Iniciantes", explora conceitos básicos como ações, fundos, renda fixa, riscos e estratégias. Carregue 3-5 fontes abertas no NotebookLM, defina objetivos, crie perguntas e teste prompts. Resultado: miniguia com resumos, glossário e prompts reutilizáveis para aprendizado via IA. Duração: 2-3h. Perfeito para novatos.
